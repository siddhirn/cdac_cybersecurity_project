# cdac_cybersecurity_project
National Cybersecurity Law for VITs is a CDAC cybersecurity project proposing a comprehensive legal and security framework for educational institutions. It focuses on cybercrime prevention, data protection, privacy rights, cybersecurity governance, incident response, and cyber awareness.
